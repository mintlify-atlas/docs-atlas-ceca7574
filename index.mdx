## Welcome to E-Dokan Store

E-Dokan is a fully-featured e-commerce storefront built on Next.js 13 App Router. It connects to a backend API to display products, categories, and promotional billboards — letting shoppers browse, filter, and purchase items with a seamless Stripe-powered checkout experience.

## Key features

<CardGroup cols={2}>
  <Card title="Product browsing" icon="magnifying-glass">
    Homepage showcases featured products with a full-width billboard. Each product card shows the image, name, price, color, and size at a glance.
  </Card>
  <Card title="Category filtering" icon="filter">
    Browse by category with size and color filters — available on both desktop sidebar and a mobile-friendly slide-out panel.
  </Card>
  <Card title="Shopping cart" icon="cart-shopping">
    Persistent cart powered by Zustand and localStorage. Add, remove, or clear items — your cart survives page refreshes.
  </Card>
  <Card title="Stripe checkout" icon="credit-card">
    One-click checkout posts selected product IDs to the backend, which creates a Stripe session and redirects the shopper to a hosted payment page.
  </Card>
  <Card title="Product detail pages" icon="image">
    Dedicated pages with image gallery, size/color display, and related products. A quick-preview modal lets shoppers inspect items without leaving the current page.
  </Card>
  <Card title="Responsive design" icon="mobile">
    Built with Tailwind CSS. Fully responsive from mobile to widescreen — navigation, filters, and product grids all adapt to screen size.
  </Card>
</CardGroup>

## Tech stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 13 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| State management | Zustand (with localStorage persistence) |
| HTTP client | Axios |
| UI components | Headless UI, Lucide React |
| Payments | Stripe (via backend API) |

<Note>
  E-Dokan Store is a **frontend-only** application. It requires a compatible backend API (such as an [ecommerce-admin](https://github.com/hempun10/edokan-store) instance) that exposes product, category, billboard, and checkout endpoints.
</Note>
